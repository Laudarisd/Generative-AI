# HPC for AI

This section turns `HPC/` into a practical handbook for high-performance computing in AI and machine learning.

It focuses on the real questions people face when they move from single-machine experiments to serious training and inference:

- how clusters are organized
- how jobs are launched
- how GPUs, memory, and storage behave
- how distributed training works
- how to debug failures and improve efficiency
