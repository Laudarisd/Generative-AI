# Cost, Efficiency, and Green Compute

HPC is not only about speed. It is also about using resources responsibly.

## Questions That Matter

- are GPUs well utilized?
- are jobs wasting hours due to bad scripts?
- is checkpointing too frequent or too expensive?
- can a smaller model solve the problem?

## Efficiency Metrics

- throughput per GPU
- utilization
- training time to target quality
- performance per watt

## Practical Levers

- mixed precision
- better batching
- better scheduling
- smaller or more efficient models
- quantization for inference

## Summary

Good HPC work is not only fast. It is efficient and defensible.
