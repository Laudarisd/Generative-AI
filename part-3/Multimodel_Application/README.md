## What This Topic Covers

A multimodal application combines more than one type of signal, such as:

- text
- images
- audio
- OCR output
- structured metadata
- tool outputs

## Example Systems

- image question answering
- document AI systems
- voice assistants
- screenshot debugging assistants
- OCR plus LLM extraction workflows
