## Why Hyperparameter Tuning Matters

Hyperparameters control training behavior without being learned directly from data.

Examples include:

- learning rate
- batch size
- weight decay
- number of epochs
- warmup ratio
- sequence length
- LoRA rank
- dropout

## Why Small Changes Matter

A poor learning rate can destroy training stability.
A poor batch size can waste hardware.
A poor sequence length can hurt both memory and quality.
