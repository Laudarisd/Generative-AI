## What LangChain Is

LangChain is an application framework for connecting prompts, models, tools, retrievers, memory-like state, and orchestration logic into one LLM application.

It is useful when your project has more than a single prompt call and starts to look like a workflow.

## Typical Things Built with LangChain

- document QA systems
- RAG pipelines
- tool-using assistants
- extraction pipelines
- agent workflows
- chat applications with structured prompts
